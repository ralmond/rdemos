What is R?

R is a language and environment for statistical computing and graphics.

It facilitates

  • data manipulation and calculations
  • graphical displays
  • statistical techniques

Available as free software

  • for UNIX platforms, Windows and MacOS
  • for download from (https://cloud.r-project.org/)
  • Current version: R 3.5.2

R Console

What is RStudio?

RStudio is an integrated development environment (IDE) for R.

It includes

  • a console
  • syntax-highlighting editor that supports direct code execution
  • tools for plotting, history, debugging and workspace management.
